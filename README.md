bnbNSA
======

The goal of this project is to create a webapp which will:
- Track club member attendance
- Keep track of their availabilty for events
- Generate statistical reports of the gathered data
- 

Api Methods For GET
1 - Getuserinfo
2 - delete
3 - add
4 - check

Api Request Example 

 1- api.php?action=getuserinfo&rfid=XXXX
 2- api.php?action=delete&rfid=XXXX
 3- api.php?action=check&rfid=XXXX
